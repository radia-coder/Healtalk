# AI Assistant Feature

## What We Built

We converted screening experience into AI assistant chat for patients.

## Main Behavior

- Warm supportive replies
- Crisis detection and crisis response
- OpenRouter as primary provider
- Fallback provider logic to improve reliability

## Routes and APIs

- Page: `/patient/dashboard/screening`
- API: `/api/screening/chat`

## Notes

System prompt and safety rules are maintained in the API route.
