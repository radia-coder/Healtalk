# History 1 — Foundation

## What We Started With

We started HealTalk as a mental health platform where patients can find therapists, book sessions, and get support.
We built the first structure with Next.js app routing, shared layouts, and role-based dashboard areas.

## Core Things We Set Up

- We set up public pages, auth pages, and dashboard routes.
- We connected PostgreSQL and Prisma for data models.
- We added NextAuth for account login and sessions.

## Why This Phase Mattered

This phase gave us a stable base so later features could be built faster without rewriting everything.
